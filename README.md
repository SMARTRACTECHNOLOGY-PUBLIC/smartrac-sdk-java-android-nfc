# smartrac-android-nfc
SMARTRAC SDK for Android NFC

## NfcNtag API

The NfcNtag API implements the command sets of following NXP chips:

 - NTAG 203
 - NTAG 210
 - NTAG 212
 - NTAG 213
 - NTAG 215
 - NTAG 216
 - NTAG 213F
 - NTAG 216F
 - NTAG I2C 1K
 - NTAG I2C 2K
